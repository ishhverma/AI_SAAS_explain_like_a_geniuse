# Explain Like a Genius 🧠

Learn any topic through the lens of history's greatest thinkers!

## Features
- **🧪 Einstein Mode**: Conceptual explanations with thought experiments
- **🎭 Feynman Mode**: Intuitive analogies and simple language  
- **💼 Buffett Mode**: Business-focused practical wisdom
- **📚 Learning History**: Track your learning journey
- **💭 Follow-up Questions**: Dive deeper into topics

## How to Use
1. Enter any topic you want to learn about
2. Choose your preferred "genius" style
3. Select depth and length preferences
4. Click "Generate Explanation"
5. Ask follow-up questions to learn more!

## Try These Topics
- Quantum physics
- Inflation
- Blockchain
- Neural networks
- Why the sky is blue
- How vaccines work

## About
This app simulates how different geniuses would explain concepts:
- **Einstein**: Thought experiments and conceptual clarity
- **Feynman**: Simple analogies and intuitive understanding  
- **Buffett**: Business principles and practical wisdom
